== Description ==

Contact Switcheroo Plugin will change the contact number on your wordpress website for a specific time duration predefined by site administrator. 

Major feature in Contact Switcheroo is to swap the contact number for a specific time Interval. Website Administrator can define or update Time Interval and Contact Numbers. 

Shortcode : [contactnumber]

== Installation ==

1.	Upload the entire Contact Switcheroo folder to the wp-contents/plugins directory.
2.	Activate the plugin in ‘Plugins’ menu in wordpress.
3. 	Once plugin is activated, you will find ‘Contact Switcheroo’ under ‘Settings’ menu.

== Frequently Asked Questions ==

= Why Plugin is displaying same number after the defined time interval? =

This problem is occured because of you are using any type of cache plugin. For better result deactivate any type of cache plugin.
